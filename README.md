# Vrinda Store Sales Dashboard

## Project Overview
Developed an interactive sales dashboard in Microsoft Excel to analyze sales performance, customer demographics, order status, and sales channels.

## Features
- Pivot Tables
- Pivot Charts
- Slicers for dynamic filtering
- Order Status Analysis
- Gender-wise Sales Analysis
- Age Group Analysis
- State-wise Sales Analysis
- Channel-wise Order Distribution

## Skills Used
- Data Cleaning
- Data Analysis
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design
- Excel Reporting

## Tools
- Microsoft Excel
